Welcome to Juniqe

Please consume the data from the following REST API:

https://localhost:3000/products-list

You will display the results showing the products images with description, prices and some other product information you want in a carousel similar to: http://v4-alpha.getbootstrap.com/components/carousel/#example

Please consider the following guidelines for your implementation:

  • compatibility > IE9
  • consider using a style preprocessor
  • responsive support
  • coded in vanilla javascript (avoid use of 3rd party libraries if possible)
  • modularised approach
  • offer a simple and intuitive navigation
  • animated transitions
  • the use of es6 is considered a plus

How to start:

  1. npm install
  2. node server.js
  3. Open https://localhost:3000
